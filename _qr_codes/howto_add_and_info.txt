how to create a new QR Code:
- create a md file in _qr_codes (e.g. example_glacier), in their you link to your targeting glacier
- create a QR Code:
	- generator: https://keremerkan.net/qr-code-and-2d-code-generator/
	- name or URL: https://goodbye-glaciers.info/qr_codes/example_glacier
	- Error correction level: high
	- block size in pixels: 20
	- save as png for use

Info about filename-prefix:
- first_signs: first collection of glaciers for testing sign layout
